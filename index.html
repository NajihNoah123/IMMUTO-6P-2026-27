Phase 2: Broken Track



There are empty slots on the track. When there are empty slots, the train won't be able to levitate or move properly. You need to fill these spots with the correct magnet for the train to pass.


You need to ensure the magnets are placed properly with the correct size and correct position. The game will want to trick you. You will need to solve some problems on the place without the magnets to find which magnet belongs there. This simulation will help you.
